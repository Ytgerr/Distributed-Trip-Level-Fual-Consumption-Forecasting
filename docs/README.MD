# Documentation Directory

This directory stores project documentation, data stories, and figures used in the report and presentation.

## Files

| File | Description |
|---|---|
| `stage1_data_ingestion.md` | Stage 1 documentation: dataset collection, PostgreSQL schema, Sqoop import, and validation. |
| `stage2_data_story.md` | Stage 2 EDA story with dashboard-ready insights and figures. |
| `stage3_ml_modeling.md` | Stage 3 documentation: feature engineering, Spark ML models, outputs, and current evaluation. |

## Figures

Stage 2 figures are stored in:

```text
docs/figures/stage2/
```

These figures mirror the generated files from `output/figures/` and are used by the Markdown data story.
